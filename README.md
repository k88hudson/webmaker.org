# webmaker.org

The bare minimum to run this:

```
npm install
cp env.dist .env
node app
```

You should be good! That said, a few things won't work (login, liking with makeapi, creating events).

In order to do that, you should run your own versions of:

https://github.com/mozilla/login.webmaker.org
https://github.com/mozilla/makeapi
